    计算几何

    在计算机图形学、CAM/CAE、游戏开发等领域，几何算法设计依赖于一系列经过验证的基础结论（定理 / 性质），这些结论能大幅简化问题建模、提升计算效率。以下是 最常用的核心几何结论，结合工程场景（如 CAM 刀具路径规划、碰撞检测、图形裁剪）详细说明，附应用场景和简化思路：一、凸多边形相关核心结论（CAM 中最常用）凸多边形是 CAM 中工件、刀具轮廓的主要形式（矩形、三角形、正多边形等），其性质是几何算法优化的关键：

    凸多边形相关

    凸多边形内 / 外点的距离极值

    结论：凸多边形外任意一点的最远点一定是多边形的顶点; 凸多边形内任意一点的最近点可能在边或顶点上，最远点 一定是顶点。
    应用场景：CAM 中刀具中心到凸多边形工件的距离计算（如碰撞检测、安全距离判断），无需遍历边的所有点，仅需计算顶点距离，效率提升 90%+。
    简化思路：例如判断球形刀具是否与凸多边形工件碰撞，仅需计算刀具中心到多边形顶点的距离，与刀具半径比较即可。

    凸多边形的分离轴定理（SAT）

    结论：两个凸多边形不相交（分离）的充要条件是：存在一条直线（分离轴），使得两个多边形在该轴上的投影无重叠。
    应用场景：CAM 中的刀具与工件碰撞检测、路径规划中的障碍物避让（如矩形刀具与凸多边形工件的碰撞判断）。
    优势：无需复杂的交点计算，仅通过投影区间比较即可判断，时间复杂度 O (n+m)（n、m 为两个多边形的顶点数），是工业级碰撞检测的首选算法。

    凸多边形的凸包性质

    结论：任意点集的凸包是包含该点集的最小凸多边形，凸包的顶点一定是原点点集中的点。
    应用场景：CAM 中不规则工件的轮廓简化（如将离散测量点拟合为凸包，减少刀具路径计算量）、刀具可达区域分析。
    工程价值：凸包能剔除冗余点，使复杂轮廓的几何计算（如距离、相交判断）更高效。

    直线与线段相关

    线段相交的快速判断

    结论：两条线段 AB、CD 相交的充要条件是：
    （1）快速排斥：两条线段的 bounding box（轴对齐矩形）无重叠，则一定不相交；
    （2）跨立实验：A、B 两点分别在 CD 线段的两侧，且 C、D 两点分别在 AB 线段的两侧（或其中一点在另一条线段上）。
    应用场景：CAM 中刀具路径的自相交检测（如避免刀具轨迹交叉）、工件轮廓的边相交判断。
    优势：避免了求解直线方程的浮点数运算，快速排斥能提前过滤大部分不相交情况，适合大规模线段集合的相交检测。

    点到直线的投影性质

    结论：点 P 到直线 AB 的最短距离，是 P 到 AB 的垂线段长度；投影点 Q 在 AB 线段上的充要条件是：Q 的参数 t∈[0,1]（设 AB 的参数方程为 A + t (B-A)，t∈[0,1]）。
    应用场景：CAM 中刀具中心到工件边缘的最短距离计算（如判断刀具是否超出加工范围）、路径的正交偏移（如生成等距刀具路径）。
    工程技巧：通过参数 t 快速判断投影点是否在线段上，避免无效计算（如 t<0 时，最近点为 A；t>1 时，最近点为 B）。

    线段的参数化表示（参数 t 的几何意义）

    结论：线段 AB 可表示为$P(t) = A + t \cdot (B - A) \quad (t \in [0,1])$
    t 的取值对应线段上的点：t=0 → A 点，t=1 → B 点，t∈(0,1) → 线段内部点，t<0 → A 点外侧，t>1 → B 点外侧。
    应用场景：CAM 中刀具路径的插值计算（如按固定步长 t 生成路径点）、线段的分段裁剪（如裁剪掉 t<0 或 t>1 的部分）。

    圆与圆弧相关

    圆 / 圆弧 与直线

    结论：圆与直线的交点最多 2 个，可通过判别式 $\Delta = r^2 - d^2$判断（d 为圆心到直线的距离，r 为圆半径）：Δ > 0 → 2 个交点，Δ = 0 → 1 个切点，Δ < 0 → 无交点。应用场景：CAM 中圆弧轮廓的刀具路径生成（如计算刀具与圆弧的切削接触点）、圆弧与直线的过渡路径规划。
    工程注意：需处理浮点数精度问题（如 Δ 接近 0 时，视为相切，避免因精度误差导致的无交点判断）。

    圆弧的弦长与圆心角关系

    圆弧的弦长 $L = 2r \cdot \sin(\theta/2)$（r 为半径，θ 为圆心角），前文已推导。
    应用场景：CAM 中圆弧的离散化（如将圆弧按固定弦长分解为线段，生成刀具路径）、圆弧长度计算

    多边形相关

    点与多边形关系

    射线法：从目标点 P 向任意方向发射一条射线，统计射线与多边形边的交点数：偶数 → 点在多边形外，奇数 → 点在多边形内（需处理射线与顶点、边重合的特殊情况）；
    环绕数法：计算多边形边围绕点 P 的总转角（顺时针为负，逆时针为正），环绕数≠0 → 点在多边形内（精度更高，适合复杂多边形）。
    应用场景：CAM 中判断刀具中心是否在工件加工区域内、路径点的有效性校验（如避免刀具走出加工范围）。
    工程选择：射线法实现简单，适合轴对齐或简单多边形；环绕数法适合复杂多边形（如带孔多边形），但计算量稍大。

    多边形面积计算

    多边形的面积计算（ shoelace 公式）结论：对于顶点按顺时针 / 逆时针顺序排列的多边形（顶点坐标 ((x_1,y_1), (x_2,y_2), ..., (x_n,y_n))），面积为：$S = \frac{1}{2} \left| \sum_{i=1}^n (x_i y_{i+1} - x_{i+1} y_i) \right|$（其中 (x_{n+1}=x_1, y_{n+1}=y_1)）

    应用场景：CAM 中工件加工面积计算（如计算切削量）、不规则轮廓的面积评估。优势：仅需遍历顶点一次，时间复杂度 O (n)，适合任意简单多边形（无自相交）。

    带孔多边形的处理（内外环分离）

    结论：带孔多边形可表示为 “外环（逆时针顺序）+ 内环（顺时针顺序）”，孔的区域通过内环与外环的 “差集” 实现。
    应用场景：CAM 中带孔工件的轮廓建模（如加工带有圆孔的矩形工件）、刀具路径的孔避让（如刀具需绕开内环区域）。

    向量相关

    向量叉积

    向量叉积的几何意义结论：二维向量 $vec{a} = (x_1,y_1))、(\vec{b} = (x_2,y_2)$的叉积 $vec{a} \times \vec{b} = x_1 y_2 - x_2 y_1$：符号：正 → $vec{b}$ 在 $vec{a})$ 的逆时针方向，负 → 顺时针方向，零 → 两向量共线；绝对值：等于以两向量为邻边的平行四边形面积（即 $vec{a}| \cdot |\vec{b}| \cdot \sin\theta$，θ 为两向量夹角）。
    应用场景：判断点的相对位置（如点 P 在直线 AB 的左侧 / 右侧）、线段相交的跨立实验、多边形顶点的顺逆时针判断。

    向量点积

    向量点积的几何意义结论：向量 $vec{a} \cdot \vec{b} = x_1 x_2 + y_1 y_2 = |\vec{a}| \cdot |\vec{b}| \cdot \cos\theta$：符号：正 → 两向量夹角 θ <90°，负 → θ> 90°，零 → 两向量垂直；应用：计算向量的投影长度（如点到直线的投影）、两向量的夹角。 工程用途：CAM 中刀具路径的方向判断（如是否与工件边缘垂直）、向量的归一化（单位向量计算）。
