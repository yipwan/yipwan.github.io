    基本认知

    日语分为罗马字，假名，日语汉字。假名又分为平假名和片假名，是同一种发音的不同形式。

    罗马字：

    • 由26个英文字母组成，标注假名发音。
    • 构成日语输入法, 比如 hou o kaimashita 表示买了书
    • 辅助假名发音。比如かわい ，发音为 kawai
    • 商标人名地名等。比如丰田 Toyota

    平假名：

    • 来源于汉字的草书
    • 用于构成单词
    • 标注汉字的发音，比如かわい 用来标注汉字“可爱”

    片假名：

    • 来源于汉字的楷书偏旁部首
    • 主要用于写外来用语，比如英文
    • 起强调作用时，可以代替平假名
    • 拟声词常用，比如dokidoki，扑通扑通

    参考：【日语五十音】一个不太正经....但能让你轻松记忆背出日语五十音的方法！_哔哩哔哩_bilibili

    日语单词分为“词”和“助词”。词是表示具体意义的独立单词，可以认为是“零件”。包括名词，动词，形容词，副词

    助词可以认为是“螺丝”，用于连接连接零件，表示零件的关系，类似英语中的介词。包括“格助词”、“副助词”、“接续助词” 和 “终助词”。

    我 + ga + 饭 + 食（我吃饭）

    句子三大类型

    • 名词句：什么是什么-> 花是植物
    • 动词句：什么怎么了 -> 花开了
    • 形容词：什么怎么样 -> 花美丽

    格助词：动词与名词之间的语义逻辑关系叫做“格”，而表现这种关系的助词叫“格助词”。

    例1：父亲 酒 家 喝 --> 父 + ga（wa） + 家 + o + 酒 + de 喝

    ga（wa）：动作主体

    o：动作场所

    de：动作受体

    基本助词

    ① 格助詞（が・を・に・で・へ・と・から・まで・より・の）

    助词 核心场景 秒记例句 中文
    が 主语、存在主体、能力对象 猫が好きです。 我喜欢猫。
    を 直接宾语 本を読む。 读书。
    に 1. 目标方向 2. 存在场所 3. 时间点 友達に会う。 / 机に本がある。 / 8時に起きる。 见朋友。 / 桌上有书。 / 8点起床。
    で 场所+手段+材料+范围 バスで学校で日本語で話す。 在校坐公交用日语说话。
    へ 移动方向（比に更强调“方向”） 東京へ行く。 去东京。
    と 共同对象、引用“说/想”的内容 母と買い物する。 / 「NO」と言う。 和妈妈购物。 / 说“NO”。
    から 起点、原因 家から出る。 / 雨から中止。 离开家。 / 因雨取消。
    まで 终点、程度 夜まで勉強する。 学习到夜里。
    より 比较基准 去年より寒い。 比去年冷。
    の 所属、属性、名词化 私の本 / 走るのが好き。 我的书 / 喜欢跑步。

    ② 副助詞（は・も・だけ・しか・くらい・ばかり・など・さえ・こそ）

    助词 核心场景 秒记例句 中文
    は 主题、对比 猫は好きだが犬は嫌い。 猫喜欢，但狗讨厌。
    も 也、甚至 日本語も英語も話せる。 日语英语都会。
    だけ 仅、只 水だけ飲む。 只喝水。
    しか〜ない 仅（否定式） 水しか飲まない。 除了水不喝别的。
    くらい/ぐらい 大约、程度 3時間ぐらい寝た。 睡了大约3小时。
    ばかり 1. 大约 2. 刚刚 3. 净是 10冊ばかり買う。 / 帰ったばかりだ。 / 故障ばかり。 买了大约10本。 / 刚回来。 / 老出故障。
    など 等等、举例 ケーキなどが好き。 喜欢蛋糕等。
    さえ 连…都 子供さえ知ってる。 连小孩都知道。
    こそ 正是…才 今日こそ勝つ！ 今天一定要赢！

    ③ 係助詞（や・とか・なり・だの）（列举）

    助词 核心场景 秒记例句 中文
    や 不完全列举 本や雑誌を読む。 读书、杂志等。
    とか 口语随意列举 お茶とかコーヒーとか。 茶啦咖啡啦。

    ④ 接続助詞（が・けど・から・ので・のに・て・ば・と・ながら・つつ・し）

    助词 核心场景 秒记例句 中文
    が/けど 逆接、单纯接续 安いが美味しい。 便宜却好吃。
    から 原因 寒いから窓を閉める。 因为冷关窗。
    ので 客观原因（比から柔和） 雨ので延期。 因雨延期。
    のに 意外、不满 一生懸命やったのに… 明明努力了却…
    て 中顿、顺序、原因 電車に乗って、本を読む。 上车后读书。
    ば 条件（假设） 春ば桜が咲く。 一到春天樱花就开。
    と 恒常条件、发现 ドアを開けると猫がいた。 一开门就有猫。
    ながら 同时做 音楽を聞きながら走る。 边听音乐边跑。
    し 并列理由 安いし美味しいし買う。 又便宜又好吃所以买。

    ⑤ 終助詞（か・ね・よ・わ・ぞ・ぜ・な・さ・っけ・なあ）

    助词 核心场景 秒记例句 中文
    か 疑问 行くか？ 去吗？
    ね 确认、共鸣 寒いね。 好冷啊，对吧？
    よ 提醒、告知 明日だよ。 是明天哦。
    わ （女性）柔和断定 知らないわ。 我不知道啦。
    ぞ （男）自我激励 行くぞ！ 出发咯！
    ぜ （男）随意 飲もうぜ。 喝吧！
    な 禁止/感叹 行くな！ / きれいな… 别去！ / 好漂亮啊…
    なあ 感叹 懐かしいなあ。 真怀念啊。

    ⑥ 間投助詞（さあ・ねえ・ええと）——口语填充

    助词 功能 例
    さあ/ねえ 催促、迟疑 「さあ、どうしよう？」
    ええと 呃… 「ええと、名前は…」

    ps： 使用顺序小贴士
    格助詞永远紧挨名词，决定“名词与动词的关系”。
    副助詞可叠在格助詞后面：
    東京から＋も → 東京からも来る。
    終助詞整句最尾，男性/女性、礼貌/粗鲁一听就知道。

    句式分析

    基礎知識から新旧ヒット曲までを通してギターを楽しく練習していこうというコンセプトで、
    词汇 ：
    基礎知識（きそちしき）＝基础知识
    新旧（しんきゅう）＝新旧
    ヒット曲（ヒットきょく）＝热门歌曲（hit song）
    通して（とおして）＝贯穿、通过（て形，表手段）
    楽しく（たのしく）＝快乐地（形容词く形，副词用法）
    練習していこう＝练习下去（～ていこう＝“去做并持续”的意志形）
    コンセプトで＝以……概念（で＝手段、方式）
