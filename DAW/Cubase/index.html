    宿主软件

    约 2476 个字 预计阅读时间 8 分钟

    记录一些宿主软件的知识与技巧

    音源轨道

    MIDI 轨（MIDI Track）

    • 作用：只记录 MIDI 事件（音符、控制器等），本身不发声，必须挂到外部硬件或机架/机内乐器上才能听到声音。 • 场景举例：写弦乐铺底：新建 8 条 MIDI 轨，分别指向同一只交响音源的 Violin1/2、Viola、Cello… 这样后期换音色、换音源都方便。

    乐器轨（Instrument Track）

    • 作用：把 VSTi 直接嵌在轨道里，一个轨道 = 一个音色 + 一个 MIDI 轨 + 一个音频输出，最直观。 • 场景举例 – 做电子鼓：直接加载 Groove Agent SE，写鼓 MIDI，混音时直接在这条乐器轨上加压缩、EQ。 – 做贝斯：插一个 Trilian Bass，写完 MIDI 后，把整条乐器轨 Freeze 成音频，省 CPU。

    采样轨（Sampler Track）

    • 作用：把任意一段音频或 Loop 拖进来，自动变成一个可演奏的采样器，可切片、包络、滤波、音高拉伸。 • 场景举例 – 把一句人声“Hey”拖进采样轨，键盘就能弹成旋律，做 Future Bass Vocal Chop。 – 鼓 Loop 拖进来切成 16 片，用 MIDI 重新编排，做出 Breakbeat。

    音频轨（Audio Track）

    • 作用：录制或导入纯波形音频，是“真声音”。 • 场景举例 – 录主唱：新建 Mono 音频轨，插 UA 610 前级录人声。 – 导入吉他 DI：把录好的 DI 音频拖进来，再插 AmpliTube 做重放大。

    混音轨道

    VCA 轨（VCA Track）

    • 作用：类似模拟调音台的 VCA 推子，用一根“总推子”同时控制若干轨的音量，但不会改变它们之间的相对平衡。 • 场景举例 – 管弦乐总控：把 Strings、Woodwinds、Brass 各编组后，再让一根 VCA 轨管“全弦乐”实时渐强渐弱。 – 影视混录：同期对白 20 条、ADR 10 条，全部受 1 条 VCA 控制，做整体微调。

    FX 通道/效果轨（FX Channel）

    • 作用：做全局效果总线，比如混响、延迟、并行压缩等，任何轨都可以 Send 到它。 • 场景举例 – 大厅混响：建一个 FX 通道挂 REVelation，所有乐器 Send 量不同，营造统一空间。 – 平行鼓压缩：鼓组各轨 Send 到 FX 通道，用 1176 All-Buttons 压成“炸裂”质感，再和干声混合。

    编组轨/Group Channel

    • 作用：把多轨先汇总成一个子混音，再做统一处理。 • 场景举例 – 鼓 Bus：Kick、Snare、OH、Room 全进 Drum Group，一条 SSL G-Bus 压缩 + 饱和，让鼓先“粘”在一起。 – 弦乐分层：Violins、Violas、Cellos 先各自乐器轨，再进一个 Strings Group，一起 EQ、压缩、加立体声宽度。

    ps：纯编曲阶段、轨道太多但暂时不混音：先建文件夹轨即可。混音阶段，文件夹轨和编组轨可以一起使用。

    命名编组轨（Group/Bus）时，核心目标是 “一眼知道它收什么、给什么、干什么”。业内已经形成了一套最简、最抗歧义的“三段式”规则：分组名-功能-可选后缀。

    命名用例参考：

    1）分组参考

    内容 缩写 例
    Drums DR DR
    Bass BS BS
    Guitars GTR GTR
    Lead Vocal VOX VOX
    Background Vox BGV BGV
    SFX SFX SFX
    Music Stem MUS MUS
    Dialogue DIA DIA

    2）功能参考

    功能 缩写 例
    压缩/粘合 comp DR_comp
    并行压缩 pcomp DR_pcomp
    空间/混响 verb VOX_verb
    延迟 dly GTR_dly
    饱和/染色 sat BS_sat
    只做路由，无处理 — DR（可省）

    3）可选名

    当同一类有多个编组，用数字或 LR 区分：

    • DR_comp 1 / DR_comp 2
    • GTR L / GTR R
    • VOX_verb s（短混响） / VOX_verb l（长混响）

    Guitars参考：

    吉他文件夹里的轨道一旦送进一条专门的“GTR Group”（编组轨），通常只在整组层面做三件事：

    1. 统一音色塑形 • EQ：低切 80–120 Hz、2–4 kHz 轻推、8 kHz 以上空气感。 • 轻饱和/磁带/箱体模拟，让多轨听起来像“同一把吉他音箱”。
    2. 统一动态 & 粘合 • 2–4 dB 的 VCA 或 Bus 压缩（1176 / LA-2A / SSL），把节奏吉他 L/R 或层层叠录“捆”在一起。 • 并行压缩：复制一路 GTR_pcomp，Blend 10–30%，增强持续感。
    3. 统一空间/深度 • 只在这组里插一个短房间或板混响（pre-delay 20–40 ms，Decay 0.8–1.2 s），让吉他轨共享同一“房间”。 • 如果还有 Lead Guitar，可在组后单独再分一路 GTR_Lead 做更长尾混响。

    总结：文件夹解决“视觉”分组，编组轨解决“音色、动态、空间”三件事，不再对单轨下手。

    管理轨道

    文件夹轨（Folder Track）

    • 作用：纯粹“收纳”轨道，可折叠/展开，移动、静音、冻结都成批处理。 • 场景举例 – 100 轨管弦：把 Winds、Brass、Perc、Strings 各放一个文件夹，工程一眼干净。 – 影视后期：把不同镜头的同期声、环境声、Foley 分文件夹，找轨不迷路。

    文件夹轨的典型使用：

    1）用例1：音乐制作

    文件夹轨名 内含轨道举例 常用总线/分组目的
    Drums Kick In / Kick Out / Snare Top / Snare Bottom / OH L\&R / Room / Perc 统一压缩、并行压缩、鼓组混响
    Bass DI / Amp / Sub Layer 侧链触发、低频 EQ
    Guitars Rhythm L / Rhythm R / Lead / Double / FX 统一箱体模拟、吉他空间
    Keys & Synth Piano / EP / Pad / Lead / Arp 统一发送混响/延迟
    Vocals Lead Vox / Double / Harmonies / Adlibs / BGs 人声总线压缩、De-esser
    FX & Risers Sweep / Noise / Reverse / Impacts 统一自动化、声像
    Submix / Print Drums Bus / Music Bus / Vox Bus → Master 最终 Stem 导出

    2）用例2：影视后期

    文件夹轨 内含轨道举例 目的
    Dialogue 主讲话 / 同期声 / ADR / Voice-over 统一降噪、EQ、压缩
    Music Score / Source / Stems 统一 Ducking（旁白压制音乐）
    SFX Foley / Hard FX / BG Ambience 统一环境混响、声像
    Mix Stems DIA Bus / Music Bus / SFX Bus → Print 交付母版/审查版本

    标尺轨（Ruler Track）

    • 作用：额外再显示一条可自定义的标尺，比如同时看“小节 & 时间码 & 采样编号”。 • 场景举例 – 影视配乐：主标尺用小节，加一条 Ruler Track 显示 SMPTE 时间码，对齐画面帧。 – Sound Design：一条标尺显示样本计数，方便精准对齐瞬态。

    标记轨（Marker Track）

    • 作用：打点、循环区、导航。可存文字和颜色。 • 场景举例 – 电影对白：在“爆炸点”“台词入”打标记，一键跳位。 – 歌曲结构：Intro、Verse、Chorus、Drop 各插标记，导出分轨时按标记循环导出。

    拍号轨（Signature Track）

    • 作用：随时改拍号、改拍子细分。 • 场景举例 – Progressive Rock：主歌 4/4、副歌 7/8、桥段 5/4，一条 Signature Track 全搞定。 – 影视配乐：画面剪辑节奏突变，用拍号轨把音乐也切成 3/4 → 4/4，保证卡帧。

    编排轨（Arranger Track）

    • 作用：用“段落块”(A、B、C…) 来实时重排整首歌，可一键做 Radio Edit、Extended Mix。 • 场景举例 – 电子舞曲：A=Intro、B=Drop1、C=Break、D=Drop2… 现场 DJ 想要“ACDCB”顺序，拖拽即可。 – 广告音乐：30 s / 15 s / 5 s 三版，用 Arranger Chain 一键导出。

    转调轨（Transposition Track）

    • 作用：整体或局部给所有 MIDI/乐器轨统一变调，不影响音频轨。 • 场景举例 – 歌手换 Key：Demo 写 C 调，歌手来了想唱 E♭，一条 Transposition Track +2，所有 MIDI 自动变。 – 管弦乐移调乐器：给整段木管 +2 半音，让 B♭ Clarinet 直接读谱。

    速度轨（Tempo Track）

    • 作用：画 BPM 曲线，做变速、Rubato、Ramp。 • 场景举例 – 抒情钢琴：现场录音有呼吸感，用 Tempo Detection 先检测，再在速度轨里微调，让管弦 MIDI 跟着真人律动。 – 广告 30 s 卡点：画面每 5 s 一个 Cut，把速度轨画成 140 → 120 → 110 阶梯，音乐自然减速。

    画面轨道

    视频轨（Video Track）

    • 作用：导入一条视频参考，支持 Mp4/Mov/Avi，帧精确播放。
    • 场景举例
    – 电影配乐：把剪辑好的 QuickTime 拖进来，边看画面边写弦乐。
    – YouTube 片头重配乐：把原片头视频导入，关掉原声，重新铺音乐。

    总结：

    • MIDI/乐器/采样/音频 是“声源”；
    • VCA/编组/FX 是“混流”；
    • 文件夹/标尺/标记/拍号/速度/转调/编排 是“管理 + 导航”；
    • 视频轨是“画面参考”。

    常用快捷键（Cubase 12）

    工程与文件
    Ctrl + N 新建工程
    Ctrl + O 打开工程
    Ctrl + S 保存
    Ctrl + W 关闭当前工程
    Ctrl + Z / Y 撤销 / 重做
    Ctrl + Q 退出 Cubase

    走带与定位（小键盘区为主）
    空格 播放 / 暂停
    小键盘 0 停止
    小键盘 . 回 00:00
    小键盘 * 录音
    小键盘 / 循环开关
    小键盘 + / – 快进 / 快退
    小键盘 1 / 2 左 / 右定位器
    小键盘 3-9 跳转到标记 3-9
    Shift + B / N 跳到上 / 下一标记
    P 把定位器套到选区
    I / O 设置插入 / 穿出录音点

    缩放与视图
    G / H 横向缩小 / 放大
    Shift + G / H 纵向缩小 / 放大
    Shift + F 缩放到全部可见
    Ctrl + 滚轮 横向缩放
    Shift + 滚轮 横向移动
    Alt + ↑ / ↓ 纵向放大 / 缩小当前轨
    Z 纵向最大化当前轨
    F2 走带条开 / 关
    F3 调音台开 / 关
    F4 VST 连接窗口
    F11 VST 乐器机架
    F12 性能表

    工具 & 编辑（数字键 1-8 必须背）
    1-8 工具依次：选择 / 范围 / 分割 / 胶水 / 擦除 / 缩放 / 静音 / 画笔
    Q 量化
    J 对齐开关
    Ctrl + A 全选
    Ctrl + D 重复选中对象
    Ctrl + K 重复循环（Repeat）
    Alt + 拖拽 复制对象
    Alt + Shift +拖拽 复制并锁定横向
    Ctrl + X / C / V 剪切 / 复制 / 粘贴
    Delete / Backspace 删除
    Shift + Delete 删除轨道

    轨道控制
    Ctrl + T 新建轨道窗口
    Ctrl + Alt + T 快速新建乐器轨
    Ctrl + Alt + R 快速新建音频轨
    S Solo 选中的轨
    M Mute 选中的轨
    Shift + S 工程设置
    Ctrl + Shift + T 新建文件夹轨
    Alt + 拖拽空白轨 复制整条轨道
    Ctrl +滚轮 切换轨道颜色

    混音与自动化
    Alt + W / R 全局 Write / Read 自动化开关（选中轨）
    Shift + F3 打开 Channel Settings
    Ctrl + Shift + S 保存当前 Mixer 快照
    Alt + 单击 Solo 独占 Solo
    Alt + 鼠标旋钮 复位默认值（任何推子/旋钮）
